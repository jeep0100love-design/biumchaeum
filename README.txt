비움과채움 V25

이번 버전 적용 내용
- 예약 시간: 08:00~18:00
- 18:00은 마지막 예약 시간으로 유지
- 19:00~22:00 예약 시간 삭제
- 관리자 로그인 상태 유지
- 새로고침하거나 앱을 닫았다 다시 열어도 관리자 로그인 유지
- 관리자 패널은 상단 관리자 버튼으로 열고 닫을 수 있음
- '관리자 로그아웃' 버튼을 눌렀을 때만 로그인 상태 해제
- 출근 시 GPS 위치 표시
- 작업중에도 GPS 위치 표시 유지
- 퇴근 시 GPS 감시 중지 및 지도 마커 삭제

기존 기능 유지
- 예약 30분 전 작업 준비중
- 예약시간 자동 작업중
- 예약 취소 시 자동 상태 해제
- 수동 작업중/퇴근 우선 처리
- 예약 버튼 영업시간 자동 제어
