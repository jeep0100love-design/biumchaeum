비움과채움 v5

업로드 방법
1. ZIP 압축을 풉니다.
2. GitHub 저장소 biumchaeum에서 Add file → Upload files를 누릅니다.
3. 이 폴더 안의 파일 전체를 업로드하고 Commit changes를 누릅니다.

관리자 기본 비밀번호
- 1004
- app.js의 ADMIN_PIN 값을 바꾸세요.

현재 동작
- 고객 예약 신청
- 30분 단위 시간
- 관리자 예약 전체 수정/삭제
- 예약 상태 변경
- 예약 가능/불가 전환
- 출근 시 위치 공유 시작
- 퇴근 시 위치 공유 완전 중지
- 공릉동 범위에서는 태릉입구역으로 고정 표시
- 공릉동 밖에서는 실제 GPS 표시
- PWA 설치

중요
- 이 버전은 GitHub Pages에서 바로 시험할 수 있도록 localStorage를 사용합니다.
- 다른 고객 휴대폰과 관리자 화면 사이의 실시간 동기화, 원격 푸시 알림은 Firebase 연결이 필요합니다.
- Firebase 연동 전에는 한 기기에서 입력한 예약이 다른 기기에 자동으로 나타나지 않습니다.
