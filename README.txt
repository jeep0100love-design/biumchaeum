비움과채움 최종 체험판 v2

추가 기능
- 예약된 시간마다 동 이름 입력
- 고객 화면에 예약 불가 시간 아래 동 이름 표시
- 예약 가능 귀여운 표정 / 예약 불가 우울한 표정
- 시간 선택 시 감사 메시지
- 예약 완료 축하 효과
- 배경음악 켜기/끄기
- 30분 단위 예약표
- 현재 위치 지도
- 홈 화면 설치형 PWA

주의
- 현재 파일은 같은 휴대폰 안에서 예약정보가 저장되는 체험판입니다.
- 여러 사람이 같은 예약표를 공유하고 관리자 휴대폰에 푸시 알림을 받으려면 Firebase 같은 서버 연결과 인터넷 배포가 필요합니다.
